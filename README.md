# Total Drama RPG - Season 1

Blog em React + Vite para acompanhar uma temporada de Total Drama RPG.

## O que tem nesta versão

- Página inicial com visual de portal da temporada
- Área de posts/notícias
- Página individual para cada post
- Página de participantes com 24 cards editáveis
- Página de eliminados com estado vazio
- Enquete visual de favoritos
- Navbar funcional com React Router
- Layout responsivo para celular

## Como rodar

```bash
npm install
npm run dev
```

## Onde editar as informações

As informações principais ficam em:

```txt
src/data.js
```

Nesse arquivo você pode alterar:

- posts
- participantes
- status dos participantes
- lista de eliminados
- números da temporada
